🥗 ご飯ほめほめボットくん (Food-Reviewer)
LINEとDify(AI)を連携させた、優しい栄養士ボットのプログラムです。

🌟 Features / 特徴
送った食事内容をAI栄養士が全力で褒めてくれます。
栄養に関する豆知識と簡単なアドバイスを返信します。
グループライン対応: グループ内では「ボットくん」と呼ばれた時だけ反応するスマート機能を搭載しています。
💬 Usage / 使い方
Individual Chat / 個別チャット: そのまま食事内容を送ってください。
Group Chat / グループチャット: "Plz call me 'ボットくん'!" （メッセージの中に「ボットくん」と入れて話しかけてね！）
🛠️ Technology / 使用技術
Google Apps Script (GAS)
Dify (Chatbot API)
LINE Messaging API
📱 LINE Account
